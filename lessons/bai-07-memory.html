Bài 07

Memory — Claude nhớ gì về bạn

Khái niệm 30 phút

Mục tiêu bài học

  • Hiểu tính năng Memory trên claude.ai hoạt động như thế nào
  • Biết Claude nhớ những loại thông tin gì và không nhớ gì
  • Nắm cách kiểm tra, chỉnh sửa và xóa memories
  • Phân biệt rõ Memory với Project context — hai thứ khác nhau hoàn toàn
  • Hiểu các khía cạnh quyền riêng tư liên quan đến Memory

Memory là gì?

Memory (Trí nhớ) là tính năng cho phép Claude tự động ghi nhớ thông tin về bạn qua nhiều cuộc hội thoại. Khác với Project context (bạn chủ động tải tài liệu lên), Memory được Claude tự động tạo ra khi phát hiện thông tin quan trọng trong quá trình trò chuyện.

Hình dung dễ hiểu

Project context giống như bạn đưa cho trợ lý một hồ sơ tài liệu để đọc — rất chi tiết, cho một mục đích cụ thể.

Memory giống như trợ lý tự ghi chú những điều quan trọng về bạn vào sổ tay cá nhân — ngắn gọn, tích lũy dần qua thời gian, áp dụng cho mọi cuộc trò chuyện.

Cách Memory hoạt động

Tự động lưu

Khi bạn trò chuyện với Claude, Claude sẽ tự phát hiện và ghi nhớ những thông tin có vẻ quan trọng. Bạn không cần ra lệnh "hãy nhớ điều này" (mặc dù bạn có thể làm vậy).

Ví dụ cuộc hội thoại:

Bạn: Giúp tôi viết email bằng tiếng Anh cho đối tác Nhật.
      Tôi là project manager ở công ty phần mềm tại Đà Nẵng.

Claude: [viết email...]

→ Claude có thể tự động lưu memory:
  "Là project manager tại công ty phần mềm ở Đà Nẵng"

Khi Claude lưu một memory, bạn sẽ thấy một thông báo nhỏ xuất hiện trong cuộc chat, cho biết Claude đã ghi nhớ điều gì. Bạn có thể click để xem chi tiết.

Tự động sử dụng

Ở các cuộc hội thoại sau (kể cả ngoài Project), Claude sẽ tự động áp dụng những gì đã nhớ:

Bạn (chat mới, tuần sau): Viết email xin nghỉ phép.

Claude: [viết email phù hợp với vai trò project manager,
        có thể nhắc bối cảnh công ty phần mềm]

→ Claude dùng memory mà không cần bạn nhắc lại nghề nghiệp

Claude nhớ những gì?

Memory thường ghi nhớ các loại thông tin sau:

Thông tin cá nhân cơ bản

  • Tên, nghề nghiệp, vị trí công việc
  • Nơi sinh sống, múi giờ
  • Ngôn ngữ ưa thích

Sở thích và phong cách

  • Bạn thích câu trả lời ngắn gọn hay chi tiết
  • Giọng văn ưa thích (chuyên nghiệp, thoải mái, hài hước)
  • Lĩnh vực quan tâm

Bối cảnh công việc và học tập

  • Dự án đang làm
  • Công nghệ và công cụ thường dùng
  • Mục tiêu dài hạn
Lưu ý

Memory lưu thông tin dưới dạng ghi chú ngắn — mỗi memory thường chỉ 1-2 câu. Đây không phải nơi lưu tài liệu dài. Nếu bạn cần Claude nhớ một tài liệu 10 trang, hãy dùng Project Knowledge thay vì Memory.

Claude KHÔNG nhớ gì?

Memory có giới hạn. Claude không nhớ:

  • Toàn bộ nội dung cuộc hội thoại: Memory chỉ lưu "điểm chính", không lưu nguyên đoạn chat
  • Thông tin nhạy cảm: Claude được thiết kế để tránh lưu mật khẩu, số tài khoản ngân hàng, thông tin y tế chi tiết
  • File bạn đã gửi: Nếu bạn gửi một file PDF trong chat thường (không phải Project), Claude không lưu file đó vào memory
  • Câu trả lời Claude đã đưa ra: Claude nhớ thông tin về bạn, không nhớ những gì chính Claude đã nói

Kiểm tra và quản lý Memory

Xem danh sách memories

Để xem Claude đã nhớ gì về bạn:

  1. Vào claude.ai
  2. Click vào biểu tượng tên bạn hoặc avatar (góc dưới bên trái)
  3. Chọn "Settings"
  4. Tìm mục "Memory"

Bạn sẽ thấy danh sách tất cả những gì Claude đã ghi nhớ, mỗi mục là một dòng ngắn.

Chỉnh sửa memory

Nếu Claude nhớ sai hoặc thông tin đã lỗi thời, bạn có thể:

  • Xóa từng memory: Click nút xóa bên cạnh mỗi mục
  • Xóa tất cả: Chọn "Clear all memories" để xóa sạch
Mẹo

Bạn cũng có thể yêu cầu Claude cập nhật memory ngay trong cuộc chat:

  • "Hãy nhớ rằng tôi đã chuyển sang làm ở Hà Nội" — Claude cập nhật memory
  • "Quên đi việc tôi đang học Python, giờ tôi chuyển sang học Go" — Claude sửa memory
  • "Đừng nhớ thông tin cá nhân của tôi nữa" — Claude dừng lưu memory cho phiên đó

Tắt Memory hoàn toàn

Nếu bạn không muốn Claude nhớ gì về bạn, vào Settings → Memory và tắt tính năng này. Khi tắt:

  • Claude không tự động lưu thông tin mới
  • Memories cũ vẫn còn (bạn cần xóa thủ công nếu muốn)
  • Mỗi cuộc chat trở nên hoàn toàn độc lập (trừ khi trong Project)

Memory vs Project Context: Khác nhau thế nào?

Memory Project Context
Ai tạo? Claude tự động tạo (hoặc bạn yêu cầu) Bạn chủ động tải lên
Phạm vi Áp dụng cho tất cả cuộc chat Chỉ áp dụng trong Project cụ thể
Dung lượng Ghi chú ngắn (1-2 câu mỗi mục) Tài liệu dài, nhiều file
Loại thông tin Sở thích, phong cách, bối cảnh cá nhân Tài liệu chuyên sâu, data, hướng dẫn chi tiết
Cập nhật Tự động khi chat Bạn phải tải lên hoặc xóa thủ công
Phù hợp cho Cá nhân hóa trải nghiệm tổng thể Một dự án hoặc mục đích cụ thể
Ví dụ kết hợp Memory và Project

Memory biết rằng bạn là giáo viên dạy Toán cấp 3, thích giải thích đơn giản, và sống ở Huế.

Project "Soạn đề thi" có tài liệu: chương trình Toán 12, bộ đề thi mẫu, quy chế thi của Sở GD&ĐT Thừa Thiên Huế.

Khi bạn mở chat trong Project này, Claude vừa biết bạn là ai (từ Memory) vừa có tài liệu chuyên sâu (từ Project) — kết quả rất chính xác và cá nhân hóa.

Quyền riêng tư và Memory

Memory lưu thông tin cá nhân, vì vậy cần quan tâm đến quyền riêng tư:

Những điều nên biết

  • Bạn có toàn quyền kiểm soát: Xem, sửa, xóa memories bất cứ lúc nào
  • Memory riêng tư cho mỗi tài khoản: Người khác dùng cùng máy tính nhưng khác tài khoản sẽ không thấy memory của bạn
  • Claude tránh lưu thông tin nhạy cảm: Được thiết kế để không ghi nhớ mật khẩu, số thẻ, thông tin y tế chi tiết
  • Có thể tắt hoàn toàn: Nếu không thoải mái, tắt Memory trong Settings
Cảnh báo

Dù Claude được thiết kế để tránh lưu thông tin nhạy cảm, không nên hoàn toàn phụ thuộc vào điều này. Tránh chia sẻ mật khẩu, số CMND/CCCD, thông tin tài chính chi tiết trong chat — dù có Memory hay không. Đây là thực hành tốt khi dùng bất kỳ AI nào.

Khi dùng tài khoản chung

Nếu bạn chia sẻ tài khoản Claude với người khác (ví dụ: tài khoản Team), hãy lưu ý:

  • Memory có thể chứa thông tin từ nhiều người
  • Xem xét tắt Memory nếu nhiều người dùng cùng tài khoản
  • Dùng Project riêng thay vì Memory để tách ngữ cảnh

Mẹo sử dụng Memory hiệu quả

Mẹo
  • Chủ động "dạy" Claude: Đầu tiên, mở chat và giới thiệu về bản thân — tên, nghề nghiệp, sở thích, phong cách làm việc. Claude sẽ lưu những thông tin này.
  • Định kỳ kiểm tra: Mỗi tháng vào Settings xem Claude nhớ gì — xóa những gì sai hoặc lỗi thời.
  • Nói rõ sở thích phản hồi: "Tôi thích câu trả lời dùng bullet points, ngắn gọn, có ví dụ cụ thể" — Claude sẽ nhớ và áp dụng.
  • Cập nhật khi có thay đổi: Đổi việc, chuyển nhà, bắt đầu dự án mới? Nói với Claude để cập nhật.

Bài tập thực hành

  1. Kiểm tra Memory hiện tại: Vào Settings → Memory trên claude.ai. Xem Claude đã nhớ gì về bạn (nếu có). Ghi lại danh sách.
  2. Giới thiệu bản thân: Mở chat mới và giới thiệu đầy đủ: tên, nghề nghiệp, sở thích, phong cách làm việc, ngôn ngữ ưa thích. Kiểm tra xem Claude lưu memory nào.
  3. Kiểm tra Memory hoạt động: Mở chat mới (khác chat ở bước 2). Hỏi Claude một câu liên quan đến nghề nghiệp mà bạn vừa giới thiệu — xem Claude có tự động sử dụng memory không.
  4. Dọn dẹp: Xóa bất kỳ memory nào không chính xác hoặc không cần thiết.

Tổng kết

  • Memory là tính năng Claude tự động ghi nhớ thông tin về bạn qua các cuộc hội thoại — không cần tải tài liệu
  • Claude nhớ thông tin cơ bản (tên, nghề, sở thích) và sở thích phản hồi — không nhớ toàn bộ nội dung chat hay file đính kèm
  • Bạn có toàn quyền kiểm soát: xem, sửa, xóa, hoặc tắt Memory hoàn toàn
  • Memory khác Project context: Memory áp dụng cho mọi chat và lưu ghi chú ngắn; Project context chỉ cho một Project cụ thể và chứa tài liệu dài
  • Kết hợp Memory + Project context = trải nghiệm cá nhân hóa tốt nhất
  • Luôn lưu ý quyền riêng tư: không chia sẻ thông tin nhạy cảm dù có Memory hay không
